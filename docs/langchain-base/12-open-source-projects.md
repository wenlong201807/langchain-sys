# LangChain + Node.js 开源项目推荐

## 官方核心项目

### 1. LangChain.js (langchainjs)

**Agent 工程平台 — LangChain 的 JavaScript/TypeScript 实现**

- **GitHub**: [https://github.com/langchain-ai/langchainjs](https://github.com/langchain-ai/langchainjs)
- **Stars**: 17,170+
- **许可证**: MIT
- **语言**: TypeScript
- **说明**: LangChain 的官方 JS/TS 版本，提供预构建的 Agent 架构和模型集成，支持 OpenAI、Anthropic、Google 等所有主流模型提供商。

---

### 2. LangGraph.js (langgraphjs)

**构建弹性语言 Agent 的图框架**

- **GitHub**: [https://github.com/langchain-ai/langgraphjs](https://github.com/langchain-ai/langgraphjs)
- **Stars**: 2,600+
- **许可证**: MIT
- **语言**: TypeScript
- **说明**: LangGraph 的 JS/TS 版本，低级 Agent 编排框架和运行时。被 Replit、Uber、LinkedIn、GitLab 等公司使用。支持确定性和 Agent 工作流的组合。

---

### 3. LangGraph.js API

**LangGraph CLI 和内存服务器实现**

- **GitHub**: [https://github.com/langchain-ai/langgraphjs-api](https://github.com/langchain-ai/langgraphjs-api)
- **Stars**: 28+
- **Forks**: 14
- **许可证**: MIT
- **语言**: TypeScript
- **说明**: 提供 LangGraph 的 CLI 工具和内存服务器实现，用于部署和管理 LangGraph 应用。

---

### 4. LangChain Azure JS

**LangChain.js 的 Azure 集成**

- **GitHub**: [https://github.com/langchain-ai/langchain-azure-js](https://github.com/langchain-ai/langchain-azure-js)
- **许可证**: MIT
- **语言**: TypeScript
- **说明**: 提供 LangChain.js 与 Azure 服务的集成支持。
