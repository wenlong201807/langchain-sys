# ThinkAgent (思智) - AI Agent 智能助手平台

## 产品愿景

**让每个知识工作者都拥有一个懂你、能干活的 AI 工作伙伴。**

## 一句话定义

ThinkAgent 是一款基于 AI Agent 技术的智能工作助手，用户可以通过自然语言对话，让 AI 帮你搜索信息、分析文档、生成内容、管理任务，并且 AI 会记住你的偏好和工作上下文，越用越懂你。

## 为什么是现在

| 趋势 | 机会 |
|------|------|
| LLM 能力突破 | GPT-5/Claude 4 级别模型已具备可靠的工具调用和推理能力 |
| Agent 框架成熟 | LangChain.js / LangGraph 提供了生产级 Agent 运行时 |
| 用户习惯养成 | ChatGPT 等产品已完成用户教育，用户期待"更能干"的 AI |
| 垂直场景空白 | 通用 AI 助手多，但"能连接工具、有记忆、可定制"的个人工作助手少 |

## 核心差异化

### vs ChatGPT / Claude

| 维度 | 通用 AI 聊天 | ThinkAgent |
|------|------------|------------|
| 工具执行 | 有限内置工具 | 开放工具生态，可自定义 |
| 记忆 | 仅对话内记忆 | 短期对话 + 长期用户画像记忆 |
| 知识库 | 无 / 有限上传 | 个人/团队知识库，RAG 检索 |
| 部署 | 仅 Web/App | Web + 小程序 + App 全端覆盖 |
| 可定制 | 低 | 支持自定义 Agent 角色、工具、提示词 |

### vs 企业级 AI 平台（Coze/Dify）

| 维度 | 企业平台 | ThinkAgent |
|------|---------|------------|
| 目标用户 | 开发者/企业 | 知识工作者个人 |
| 上手成本 | 需要配置工作流 | 开箱即用，对话即服务 |
| 价格 | 企业级定价 | 免费增值模式 |
| 体验 | 工具感强 | 伙伴感，有温度 |

## 目标市场

### 第一阶段 (MVP)：个人知识工作者

- 自由职业者、内容创作者、研究人员、产品经理、程序员
- 预估 TAM：中国市场 5000 万知识工作者

### 第二阶段：小团队协作

- 3-20 人小团队，共享知识库和 Agent
- 预估 SAM：500 万小团队

### 第三阶段：企业定制

- 企业私有化部署，对接内部系统
- 预估 SOM：50 万企业客户

## 核心技术栈选型

| 层级 | 技术 | 理由 |
|------|------|------|
| AI 运行时 | LangChain.js + LangGraph | 成熟的 Agent 框架，原生 TS 支持 |
| 后端 | Node.js (Fastify/Express) | 与 LangChain.js 同生态，团队精通 |
| Web 前端 | React (Next.js) | 生态丰富，SSR 支持，SEO 友好 |
| 移动端 | UniApp | 一套代码，小程序 + App 全覆盖 |
| 数据库 | PostgreSQL + Redis | 可靠持久化 + 高性能缓存 |
| 向量数据库 | Pinecone / pgvector | RAG 知识库检索 |
| 对象存储 | 阿里云 OSS / S3 | 文档、图片存储 |
